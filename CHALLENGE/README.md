# React/Next Forberedelses-utfordring (Challenge)

Du kan begynne på denne oppgaven når du har fullført de andre 4 seksjonene.

## Oppgave: Produktkatalog med filter

Lag en liten webapp (bruk [index](./index.html) filen):

- Laster produktdata fra `products.json` med `fetch` (async/await).
- Rendrer produktene som kort i DOM (navn, pris, kategori, bilde).
- Har et filter (dropdown) som lar brukeren filtrere på kategori (inkl. “Alle”).
- Er responsiv (1 kolonne på mobil, 2–3 på desktop via media queries).
- Bruker semantisk HTML og tilgjengelighets-basics (alt-tekst, label på filter).

## Data (forslag)

Du kan bruke en JSON-array med minst 5 produkter og minst 2 kategorier.

## Rubrikk (pass/fail)

**Bestått** når:

- Data lastes korrekt, liste rendres, filter fungerer.
- Layout funker på mobil og desktop uten horisontal scrolling.
- Koden er ryddig, moderne JS, og uten blokkende console errors.

Legg ved skjermbilder (desktop + mobil) og evt. console-output i PR.
