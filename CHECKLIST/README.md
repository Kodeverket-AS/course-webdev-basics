# React/Next readiness checklist

Denne listen kobler ferdigheter du lærte i dette repoet til typiske ting du gjør i React/Next.js.

## HTML
- [ ] Semantiske elementer → JSX består av HTML-lignende tags. God struktur gir bedre tilgjengelighet og enklere vedlikehold.
- [ ] Forms + label/required → skjema i React (kontrollerte inputs) bygger på samme prinsipper.

## CSS
- [ ] Box model → spacing og størrelser i komponenter.
- [ ] Flex/Grid → layout i komponentbasert UI.
- [ ] Responsivitet → mobil first UI i webapper.

## JavaScript
- [ ] let/const, arrow, template literals → standard i moderne React kode.
- [ ] Destructuring → `const [state, setState] = useState()` og props destructuring.
- [ ] Spread/rest → kopiere state immutably, sende props.
- [ ] map/filter/reduce → rendre lister og filtrere state.
- [ ] import/export → hver React fil bruker modules.
- [ ] async/await + fetch + JSON → datahenting i Next/React.

## Tooling
- [ ] npm scripts → `npm run dev/build/start` i Next.
- [ ] DevTools → feilsøking av DOM/CSS/Network/Console.
- [ ] Git + PR → samarbeid og innlevering i team.
