# React/Next readiness checklist

Denne listen kobler ferdigheter du lærte i dette repoet til typiske ting du gjør i React/Next.js.

## HTML
- [x] Semantiske elementer → JSX består av HTML-lignende tags. God struktur gir bedre tilgjengelighet og enklere vedlikehold.
- [x] Forms + label/required → skjema i React (kontrollerte inputs) bygger på samme prinsipper.

## CSS
- [x] Box model → spacing og størrelser i komponenter.
- [x] Flex/Grid → layout i komponentbasert UI.
- [x] Responsivitet → mobil first UI i webapper.

## JavaScript
- [x] let/const, arrow, template literals → standard i moderne React kode.
- [x] Destructuring → `const [state, setState] = useState()` og props destructuring.
- [x] Spread/rest → kopiere state immutably, sende props.
- [x] map/filter/reduce → rendre lister og filtrere state.
- [x] import/export → hver React fil bruker modules.
- [x] async/await + fetch + JSON → datahenting i Next/React.

## Tooling
- [x] npm scripts → `npm run dev/build/start` i Next.
- [x] DevTools → feilsøking av DOM/CSS/Network/Console.
- [ ] Git + PR → samarbeid og innlevering i team.
